## `<Skeleton>`

```ts
type SkeletonProps = {
  // Required
  loading: boolean;
  children: ReactNode;
  baseColor: string;
  highlightColor: string;

  // Optional — visual
  animation?: "shimmer" | "pulse" | "none"; // default 'shimmer'
  transition?: boolean | number; // ms; default 300, false disables
  minShowDuration?: number; // ms; default 500
  delayShowDuration?: number; // ms; default 100

  // Optional — extension hooks
  renderBone?: RenderBoneFn;
  classify?: ClassifyFn;
  refineBones?: RefineBonesFn;

  // Optional — observability
  onMeasured?: (bones: ReadonlyArray<BoneRect>, tree: BoneNode) => void;

  // Optional — layout / a11y
  style?: StyleProp<ViewStyle>;
  accessibilityLabel?: string; // default "Loading"
};
```

### Required props

- **`loading`** — when `true`, render the skeleton; when `false`, render children as-is. Transition and sticky duration are handled internally.
- **`children`** — the real component. Pass valid data (real or mock) so layout can be measured. Children always render while loading; they're hidden via `opacity: 0`.
- **`baseColor`** — background color of each bone.
- **`highlightColor`** — sweep color for shimmer, pulse target for pulse.

### Optional visual props

- **`animation`** — `'shimmer'` (default) is a translating gradient sweep; `'pulse'` is an opacity oscillation; `'none'` renders static bones (useful for test snapshots).
- **`transition`** — milliseconds for the fade-out on `loading=false`. `true` uses 300ms; `false` disables the fade entirely.
- **`minShowDuration`** — milliseconds. Once the skeleton appears, it stays for at least this long to prevent flicker.
- **`delayShowDuration`** — milliseconds. Loading must stay true for this long before the skeleton appears.

### Extension hooks

- **`renderBone`** — replaces the default `<Bone>` per bone. See [renderBone](/react-native-fabric-shimmer/extension/render-bone/).
- **`classify`** — overrides the default fiber classifier. See [classify](/react-native-fabric-shimmer/extension/classify/).
- **`refineBones`** — transforms the IR tree before flattening. See [refineBones](/react-native-fabric-shimmer/extension/refine-bones/).

### Observability

- **`onMeasured`** — called with the flattened `BoneRect[]` and refined `BoneNode` tree each time measurement completes. Useful for debug overlays, `dumpTree(tree)`, analytics, or memoization.

## `<Bone>`

```ts
type BoneProps = {
  rect: BoneRect;
  progress: SharedValue<number>;
  baseColor: string;
  highlightColor: string;
  animation: AnimationKind;
};
```

The default bone renderer. Exposed so custom `renderBone` implementations can
delegate the default visual back in for bones they don't want to override.
Props are flat scalars (rather than a nested `ctx` object) so React Compiler
can memoise the body across `<Skeleton>` re-renders when nothing
bone-relevant has changed.

When delegating from a `renderBone` callback, pick the four ctx fields
explicitly:

```tsx
renderBone={(rect, ctx) =>
  rect.kind === "image" ? (
    <MyImageBone rect={rect} ctx={ctx} />
  ) : (
    <Bone
      rect={rect}
      progress={ctx.progress}
      baseColor={ctx.baseColor}
      highlightColor={ctx.highlightColor}
      animation={ctx.animation}
    />
  )
}
```

### Container vs leaf rendering

The default `<Bone>` distinguishes between two kinds of rect that the
classifier emits:

- **Container** (`rect.kind === "container"`) — a wrapping View with its
  own background and/or border in the original source (e.g. a card with
  `borderRadius` and a `surface` background). The default Bone renders
  these **statically**, using the source's captured `backgroundColor` /
  `borderColor` / `borderWidth` verbatim. The card outline stays present
  as visual context; the shimmer or pulse animation doesn't apply to it.
- **Leaf** (`rect.kind === "text" | "image" | "view"`) — the actual
  placeholder for content. These animate (shimmer / pulse / none),
  filled with `baseColor` and outlined with a hairline `highlightColor`
  border for definition.

The split makes the loading state read as "an empty version of the real
card with shimmering content inside" rather than "a fully shimmering
blob." Override either by classifying differently with `classify`, or
by replacing the visual with `renderBone`.

## Types

```ts
type BoneKind = "text" | "image" | "view" | "container";

type BoneRect = {
  readonly x: number;
  readonly y: number;
  readonly width: number;
  readonly height: number;
  readonly borderRadius: number | "50%";
  readonly kind: BoneKind;
  readonly backgroundColor?: string;
  readonly borderColor?: string;
  readonly borderWidth?: number;
};

type BoneContext = {
  readonly progress: SharedValue<number>;
  readonly baseColor: string;
  readonly highlightColor: string;
  readonly animation: "shimmer" | "pulse" | "none";
  readonly index: number;
  readonly total: number;
};

type BoneNode = {
  readonly id: string;
  readonly type: string;
  readonly classification: "leaf" | "container" | "transparent";
  readonly rect: {
    readonly x: number;
    readonly y: number;
    readonly width: number;
    readonly height: number;
  };
  readonly style: {
    backgroundColor?: string;
    borderRadius?:
      | number
      | { topLeft?: number; topRight?: number; bottomLeft?: number; bottomRight?: number };
    borderColor?: string;
    borderWidth?: number;
  };
  readonly children: ReadonlyArray<BoneNode>;
};

type FiberNode = {
  readonly type: string | Function | null;
  readonly memoizedProps: unknown;
  readonly stateNode: unknown;
  readonly child: FiberNode | null;
  readonly sibling: FiberNode | null;
  readonly return: FiberNode | null;
};

type FiberClassification = "leaf" | "container" | "transparent" | "skip";

type RenderBoneFn = (rect: BoneRect, ctx: BoneContext) => ReactNode;
type ClassifyFn = (fiber: FiberNode) => FiberClassification;
type RefineBonesFn = (tree: BoneNode) => BoneNode;
type OnMeasuredFn = (bones: ReadonlyArray<BoneRect>, tree: BoneNode) => void;
```

## IR helpers

Exported for use inside `refineBones`:

- `walk(tree, visitor)` — depth-first pre-order traversal; visitor receives `(node, path)`.
- `find(tree, predicate): BoneNode | null` — first match.
- `findAll(tree, predicate): BoneNode[]` — all matches, in pre-order.
- `hide(tree, target): BoneNode` — returns a new tree with the target subtree removed.
- `merge(tree, targets: BoneNode[]): BoneNode` — returns a new tree with the targets replaced by one bone at their union rect.
- `union(rects): Rect | null` — bounding rect utility.

## Default classifier

Exported so custom `classify` functions can delegate for nodes they don't want to override:

```ts
import { defaultClassify, type ClassifyFn } from "react-native-fabric-shimmer";

const classify: ClassifyFn = (fiber) => {
  const t = fiber.type;
  if (typeof t === "function" && (t as { displayName?: string }).displayName === "MyIcon")
    return "leaf";
  return defaultClassify(fiber);
};
```
