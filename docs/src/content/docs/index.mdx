## What You Get

`react-native-fabric-shimmer` builds skeletons from the component you already
own. No parallel loading layout. No code generation. No JSON list of rectangles
that goes stale after the next design pass.

- **Measured at runtime** through React Native Fabric.
- **Shimmer, pulse, or static bones** with reduce-motion-aware Reanimated worklets.
- **A small IR** for hiding, merging, and replacing measured bones.
- **Native-first validation** through the example app; web Storybook is only a deterministic visual preview.

## First Look

```tsx
import { Skeleton } from "react-native-fabric-shimmer";

<Skeleton loading={user === undefined} baseColor="#e4e4e7" highlightColor="#f4f4f5">
  <UserCard user={user ?? PLACEHOLDER_USER} />
</Skeleton>;
```
