## No Bones Appear

Start with the platform contract. The library needs Fabric host nodes and
`global.nativeFabricUIManager.measureLayout`; web previews and old architecture
native builds cannot provide them.

In development, warnings use the `[fabric-shimmer]` prefix.

### `containerRef.current is null`

The outer measurement container is not mounted yet. This usually means the
component unmounted during the loading transition or the layout event fired
before the ref attached.

Fix: keep the `Skeleton` mounted while loading and avoid conditionally removing
the wrapper around it.

### `containerRef has no fiber instance`

The ref does not expose React Native's internal fiber handle. The usual cause is
view flattening or a non-host wrapper.

Fix: keep `collapsable={false}` on the native wrapper. The library already does
this for its own outer and content views; add it to surrounding portal/modal
boundaries if needed.

### `Fabric state node not found on container`

The app is not running on the Fabric renderer for that subtree.

Fix: enable New Architecture, rebuild native binaries, and verify the screen is
not inside a renderer boundary that falls back to Paper.

### `global.nativeFabricUIManager is not available`

The host looked Fabric-shaped, but the global Fabric manager is missing.

Fix: rebuild with New Architecture enabled. In Expo, set `newArchEnabled: true`
and use a native build or dev client, not an environment that lacks the native
runtime.

### `No measurable bones found`

The component rendered, but the classifier found no text, image, text input, or
distinctive surface.

Fix: pass realistic placeholder data so content exists, or provide `classify`
for custom host components that should become bones.

## Debugging Shape

Use `onMeasured` with `dumpTree`:

```tsx
<Skeleton
  loading
  baseColor="#e4e4e7"
  highlightColor="#f4f4f5"
  onMeasured={(_bones, tree) => console.log(dumpTree(tree))}
>
  <UserCard user={PLACEHOLDER_USER} />
</Skeleton>
```

If the tree is correct but too noisy, use `refineBones` with `hide`, `merge`,
or a custom transform.

## Shimmer Looks Choppy

- Ensure `react-native-worklets/plugin` is last in `babel.config.js`.
- Confirm Reanimated 4 is installed once in the app.
- Use `animation="none"` for deterministic screenshots.
- Use the native profiler for runtime animation issues; Storybook fixture
  snapshots do not exercise native worklets.
